Katana Apprentice

Katana Apprentice was a group project created in a semester at QUT. It is a VR game where the user gets

to clean, sharpen and test katanas for clients.

For the project, I created the Game Design Document and was mainly the designer for the project.

The other group members did most of the programming for cleaning the katana and customer.

I also made an oil bottle object in blender which was used in the game.

I also worked on the UI interaction for the main menu and the cleaning off the dirt from the Katana.

Overall, this project was great to work on as it allowed me to gain the skills and understanding

of how to create a VR game using Unity and C#. These skills would allow me

to work on a VR project with an external client for my capstone project.
