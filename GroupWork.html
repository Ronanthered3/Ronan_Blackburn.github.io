Painting Mechanic

The game from which this mechanic was made for was a group project with an external party who gave us the task.

The main idea for the game was a house building game similar to The Sims and House Flipper.

The painting mechanic would allow the player to make the house more personal by being

able to paint the walls using the provided colours.

As seen in the Gif above, the painting was done using Unity's particle system. This allowed it to look like spray

paint and allowed the player to use it however they wanted.
